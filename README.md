This app is a web application built with Node and Express that allows you to shortern long URLs.

## Dependencies

- Node.js
- Express
- EJS
- bcrypt
- body-parser
- cookie-session
- morgan
- nodemon
- chai
- mocha

## Get Started!

1. Install all dependencies 
2. Run the web server locally using the `node express_server.js` command.
